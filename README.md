# ywwのAI-智能伴侣

一个基于 DeepSeek API 的 AI 聊天应用，可以创建多个角色，每个角色有自己的性格和好感度系统。

## 快速开始

### 安装依赖
```bash
pip install streamlit openai
```

### 运行
```bash
streamlit run main.py
```

### 配置 API Key
- 方式1：设置环境变量 `DEEPSEEK_API_KEY`
- 方式2：启动后在页面输入框手动输入
- 具体请参考AI伴侣_帮助文档.docx
## 主要功能

- **多角色管理**：创建、切换、删除不同的 AI 角色
- **好感度系统**：0-100分，影响 AI 的对话态度
- **智能评估**：每轮对话后 AI 自动评估好感度变化
- **数据保存**：聊天记录自动保存到 session 文件夹

## 好感度等级

| 分数 | 状态 | AI态度 |
|------|------|--------|
| 0 | 拉黑 | 删除角色 |
| 1-20 | 冷淡 | 敷衍回复 |
| 21-40 | 初识 | 礼貌疏离 |
| 41-60 | 友好 | 正常聊天 |
| 61-80 | 亲密 | 温柔关心 |
| 81-100 | 热恋 | 撒娇表白 |

## 项目结构

```
AI智能伴侣/
├── main.py              # 主程序
├── session/             # 角色数据（JSON文件）
└── resource/            # 图片资源
```
